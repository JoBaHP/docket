FAQ

  • How can I use it?
    • Just press the "Create new checklist" button. Then in your new checklist "Add groups" and select which checklist groups are relevant for your project.
      You can add new custom items, modify presets from template or delete them.
  • Is a checklist persistent?
    • Yes, each checklist get a custom url which can be shared with friends and colleagues.
  • Can I change templates?
    • The templates are actually hardcoded jsons in github repository . Please create a pull request (with a comment). I'll be happy to accept it, if it makes a sense.
  • Is the checklist private
    • No and Yes, each checklist has unique url. It is so complex, that there is almost zero chance someone guess you checklists url.
  • Can more users work on same checklist at the same time?
    • Yes, it supports realtime updates via websockets. You will changes of other users in real time (changes are submited once input is blurred, not on every keystroke)
  • Why should I register/log in?
    • When logged in, all visited checklists are recorded and shown in main page. So you can easily access them later from othe computer.
  • Why I don't see my checklist from other computer?
    • Visited checklists are stored in localStorage, hence related to computer and browser were you cretated or visited them. Try to register to keep your checklists across machines.
